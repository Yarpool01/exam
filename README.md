# Exam System

Система проведения экзаменов с билетами разной сложности.

## Деплой на Railway

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Создайте проект → Deploy from GitHub repo
3. Выберите этот репозиторий
4. В настройках проекта добавьте переменные:
   - `ADMIN_PASSWORD` — ваш пароль для админки
5. Дождитесь деплоя
6. Откройте URL из Dashboard → админка

## Локальный запуск

```bash
pip install -r requirements.txt
python main.py
```

Откройте http://localhost:8000/admin

## Структура

- `/admin` — админ-панель (требует пароль)
- `/{token}` — страница студента
- CSV импорт поддерживается через админку
